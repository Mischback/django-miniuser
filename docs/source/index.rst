django-miniuser
===============

**django-miniuser** is a pluggable Django application to provide a complete
usermanagement workflow. This includes *registration*, *login*/*logout* and
*password reset*. It integrates fully into Django's admin interface and reuses
Django's permission system.

Notable Features
----------------

* replaces Django's default user model
* provides views for *login*, *logout*, *password reset* and *registration*
* compatible with Python 2.7, 3.4, 3.5, 3.6
* compatible with Django 1.10, 1.11, 2.0
